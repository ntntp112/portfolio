How unpopular/popular is Donald Trump?

  • Posted on March 01, 2019

How did I try to replicate this visualization?

First, I realize that there are two area graphs which are approval high/low and disapproval high/low. Then I tried to bring 4 dimensions into the visualization as 4 area graphs and it did not work. I tried to switch to 4 lines and it did not work too because it supposes to be an area graph.

Second, I tried to visualize only 2 dimensions which are approval high and approval low with a dual synchronized axis and I could make one area which is the green only since dual-axis graph only allows 2 dimensions. The trick here is to place a low dimension on top of the high dimension and color the low one with white so it matches with the background of the visualization. Finally set the opacity to 100%.

However, that was just only approval area. Since the dual-axis graph only allows 2 dimensions, I tried to use Tableau Prep to transform the data source and use pivot function to convert necessary dimensions into two dimensions which are area_type and area_value. Next, the area of orange and green in the visualization is shown using only one dimension named "area_value" which the same trick above and "area_type" is used as color.

Following the same technique above, I used pivot function to make another 2 dimensions which are "line_type" and "line_value". Lines, which is shown in the visualization, are leveraged from "line_value" dimension and colored by "line_type" dimension.

There is one information missing compare to the origin visualization. I tried many add circles/dots into the visualization but no success.
